# 部署检查清单

## 上传服务器前

- 已包含 `dist/` 构建产物。
- 已包含 `public/data/` 与 `dist/data/` 数据文件。
- 未包含 `node_modules/`。
- 未包含 `.git/`。
- 未包含 `.Rhistory`、临时日志和本机缓存。
- `docker-compose.yml` 端口符合服务器要求。

## 启动后

- `docker compose ps` 显示服务为 `running` 或 `healthy`。
- `http://服务器IP:8080/healthz` 返回 `ok`。
- `http://服务器IP:8080/red-tide-risk/` 可以打开网站。
- 风险地图、模型评估、特征分析、相关分析等页面图表能够正常显示。
- 浏览器控制台没有 `404 data/*.json` 错误。
